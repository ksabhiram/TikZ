# TikZ
Maintaining an open-access TikZ schematic repository for  experimental physics instrumentation, inviting community  contributions to reduce figure reproduction overhead in research.

## TikZ Experimental Physics Repository

A growing, community-driven repository of TikZ schematics for 
experimental physics setups — free and open for everyone.

Finding clean, ready-to-use TikZ figures for experimental physics 
is frustratingly difficult. This repository aims to change that.

Upload your schematics, use others', and help build the largest 
open collection of experimental physics TikZ figures available.

### Contribute
Have a schematic? Submit a pull request. 
Every figure shared saves someone else hours of work. A breif Preview can be seen here.
